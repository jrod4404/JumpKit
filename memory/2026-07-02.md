### JumpKit Auth Bug — Login Flash/Redirect Loop (CRITICAL)
**Bug:** After installing prod DMG on Mac, login flashes home page then redirects back to sign-in. Affects all accounts, all Mac machines.

**Root cause:** Custom async storage adapter in `supabase/client.js` used an in-memory Map as fallback when `safeStorage` (Electron's encrypted keychain) was unavailable or failed. The in-memory Map dies when navigating from `index.html` → `app.html`, so Supabase finds no session and redirects back to login.

On non-notarized builds, macOS may prompt keychain access permission; if that fails silently, safeStorage fails and the in-memory Map was the only fallback.

**Fix applied:** Changed `setItem` to fall back to `localStorage` (persistent across page navigation) instead of the in-memory Map. `getItem` also updated to always return localStorage value regardless of whether safeStorage migration succeeded.

**File:** `Topics/JumpKit/app/supabase/client.js`

**Status:** Fix applied, new prod DMGs rebuilt (arm64 + x64). Jeff to install new arm64 DMG and verify login persists.

### JumpKit Auth Bug — Login Flash/Redirect Loop (CRITICAL)
**Bug:** After installing prod DMG on Mac, login flashes home page then redirects back to sign-in. Affects all accounts, all Mac machines.

**Root cause:** Custom async storage adapter in `supabase/client.js` used an in-memory Map as fallback when `safeStorage` (Electron's encrypted keychain) was unavailable or failed. The in-memory Map dies when navigating from `index.html` → `app.html`, so Supabase finds no session and redirects back to login.

On non-notarized builds, macOS may prompt keychain access permission; if that fails silently, safeStorage fails and the in-memory Map was the only fallback.

**Fix applied:** Changed `setItem` to fall back to `localStorage` (persistent across page navigation) instead of the in-memory Map. `getItem` also updated to always return localStorage value regardless of whether safeStorage migration succeeded.

**File:** `Topics/JumpKit/app/supabase/client.js`

**Status:** Fix applied, new prod DMGs rebuilt (arm64 + x64). Jeff to install new arm64 DMG and verify login persists.
