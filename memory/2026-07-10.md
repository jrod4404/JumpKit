## PostKit — Image Generation Fixes
Jeff reported two bugs + one change request:
1. **Text cut off / not centered** in generated images
2. **Text-to-replace not replaced** in some instances
3. **JumpKit logo should be 50% bigger** in generated images

### Fixes Applied (image-renderer.js)
1. **Text replacement bug**: Root cause was using `String.replace()` with regex, where `$` characters in replacement strings get interpreted as backreferences. Switched to `split().join()` pattern — eliminates all regex substitution bugs.
2. **Text centering**: Widened safe areas for all templates (more vertical room), reduced maxChars per line to prevent overflow with proportional fonts.
3. **Logo 50% bigger**: Changed from `width=180 height=75` to `width=270 height=112`, repositioned x/y accordingly.
