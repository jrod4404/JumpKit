**Save Results reliability fix (critical)**

- Fix: both `_saveReleaseSection` and `_autoSaveAllSections` now read from BOTH `_jkTestResults` + `_jkWinTestResults` independently, with "prefer meaningful state" merge — never overwrites real pass/fail with not-run

### Lessons
- Electron caches JS at launch — code changes require app restart to take effect
- Always use polling to wire Modal buttons (Modal queue delays render past static setTimeout)
- `<tr style="A" style="B">` — browser keeps only FIRST style attr; must merge into one
- JS spread order matters: `{...obj, key: val, ...obj2}` where obj2 has `key` will clobber `val`
