### Bugs fixed
1. **send-feedback edge function was never deployed** — file existed at `supabase/send-feedback.js` root but had no `supabase/functions/send-feedback/` folder. Created and deployed.
2. **CORS blocked Electron** — edge function had `Access-Control-Allow-Origin: https://jumpkit.app` but Electron apps use `null` (file://) origin. Changed to `*` (safe, bearer token auth still enforced).
3. **`window.SUPABASE_ANON_KEY` was undefined** — `config.js` uses `const` which doesn't attach to `window`. `app.js` and `auth.js` were referencing `window.SUPABASE_ANON_KEY` → sending `Bearer undefined` → 401 Invalid JWT. Fixed to `SUPABASE_ANON_KEY` (no window prefix). Same bug was in `auth.js` for `window.SUPABASE_URL` on the send-welcome call.
